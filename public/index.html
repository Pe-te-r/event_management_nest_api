🎉 Event Management System API

This API allows you to manage events, users, registrations, feedback, and more — built for organizers, attendees, and admins.

📘 View Swagger Docs 🌐 GitHub Source

📌 API Modules

  • /auth — Login, Register, Token Refresh
  • /users — CRUD users (admin only)
  • /events — Create, Update, Delete (admin/organizer), View (public)
  • /events_registration — Register for events, View registrations
  • /feedbacks — Submit/view feedback for events

🔐 Access Control

  • Admin — Manage users, events, and feedback
  • Organizer — Create and manage own events
  • Authenticated Users — Register and give feedback
